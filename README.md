## Lightweight Book Recommender
基于 Kaggle 7k Books 数据集的轻量级智能图书推荐系统
本项目基于 Kaggle 的“7k Books with Metadata”数据集（约 7000 本图书元数据），通过系统化的数据清洗、文本向量化与情感分析，构建了一个轻量、高效、可交互的图书推荐平台。前端采用 Gradio 实现，用户可以轻松通过关键词搜索、自然语言描述或情绪偏好获取个性化推荐，同时查看图书封面、摘要、评分及推理依据。
核心亮点

- 智能语义推荐：无需精确书名，一句描述即可匹配相似图书。
- 情绪驱动排序：根据用户当前心情（如 joy、sadness）优先推荐情感匹配的书籍。
- 完整信息填充：针对缺失描述的图书，自动推断主题与情绪标签，确保每本图书均具备推荐基础。
- 轻量部署：纯本地运行（结合 OpenAI/Hugging Face API），易于扩展与分享。

数据预处理与分析

- 对数据集进行全面清洗，包括缺失值处理与字段筛选。
- 可视化分析：缺失字段分布、出版年份统计、页数-评分相关性热力图等，帮助识别关键推荐特征。
- 筛选描述长度 > 35 词的图书，确保语义质量。
- 对于缺失描述的图书：
- 使用 Hugging Face Zero-shot Classification 模型推断主题标签。
- 生成简要情绪标签（surprise / neutral / sadness / anger / fear / joy / disgust）。


技术实现

- 向量化：采用 OpenAI 的 text-embedding-3-small 模型对图书描述进行嵌入，实现高效的 similarity_search 语义检索。
- 情绪分析：结合 Hugging Face Sentiment Classification 管道，辅助情绪偏好推荐（对匹配情绪的图书进行结果排序）。
- 推荐逻辑：融合语义相似度与情绪匹配，提供多样化、个性化的图书列表。

交互界面（Gradio）
一个直观、响应迅速的前端界面，支持以下功能：

- 关键词搜索：输入书名、作者或主题快速查找。
- 语义推荐：输入任意句子（如“我想读一本关于太空冒险的励志故事”），系统返回最相似的图书。
- 情绪驱动推荐：选择情绪标签（如 joy 或 sadness），获取情感共鸣的书籍，并按匹配度排序。
- 丰富展示：每条推荐自动显示封面图片、评分、摘要及相似度/情绪推理说明。
